# SIGNOVA - Sistema de Gestión de Usuarios

## 📌 Descripción
Proyecto desarrollado en Java Web utilizando Servlets, JSP y MySQL.  
Permite gestionar usuarios mediante operaciones CRUD (Crear, Leer, Actualizar y Eliminar).

Este proyecto hace parte de la evidencia:
**GA7-220501096-AA2-EV02 - Módulos de software codificados y probados**

---

## 🛠 Tecnologías utilizadas
- Java
- Servlets (Jakarta)
- JSP
- MySQL
- Apache Tomcat
- HTML5 / CSS3

---

## ⚙️ Funcionalidades
- Registrar usuario
- Listar usuarios
- Buscar usuario
- Actualizar usuario
- Eliminar usuario
- Prueba por consola (run-prueba.bat)
